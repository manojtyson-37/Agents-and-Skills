# Working Preferences

**Communication Style:** Caveman mode (full) — terse, direct, no articles/filler/pleasantries.

## Decision Framework

1. Accuracy
2. Reliability
3. Scalability
4. Long-term maintainability
5. Minimum rework

- **Status updates:** Short, outcome-focused
- **Context:** Assume prior knowledge from memory files; don't repeat

- Validation done
- Risks identified
- Open items documented
- Next steps clear
